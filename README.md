## üìö Learning Topics
The course covers various fundamental topics essential for understanding object-oriented programming. These include:
- **Abstraction:** Simplifying complex reality by modeling classes based on the essential properties.
- **Classes & Objects:** Understanding the blueprint of creating objects and instances.
- **Inheritance:** Learning how derived classes can inherit methods and properties of base classes.
- **Polymorphism:** Exploring how to create methods that can use objects of different types.
- **Encapsulation:** Protecting an object's state by restricting direct access.
- **Interfaces:** Defining methods that must be created within any class that implements the interface.
- **Abstract Classes:** Understanding classes that cannot be instantiated on their own.
